# MyFirstSmartContract

## Steps to Set Up the Development Environment
1. Installed MetaMask and set up an Ethereum wallet.
2. Accessed Remix IDE and created a workspace.
3. Created a new Solidity file, `SimpleStorage.sol`.

## Writing, Compiling, and Deploying the Smart Contract
- Created a simple smart contract to store a favorite number using a `uint256` variable.
- Compiled and deployed the contract locally using Remix VM (Cancun).
